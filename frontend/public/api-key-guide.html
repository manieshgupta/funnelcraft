API Key Generation Guide

Funnelcraft operates on a secure **Bring Your Own Key (BYOK)** model. Follow the instructions below to get your AI API credentials.

G

Google Gemini API

High velocity and intelligence, recommended provider for fast drafts.

Open Google AI Studio ↗
1

Go to Google AI Studio and log in using your Google account.

2

Click the **"Get API key"** button on the top left navigation sidebar, or go directly to https://aistudio.google.com/app/apikey.

3

Click the **"Create API key"** button. Choose to bind your key to a Cloud project or create a default key.

4

Copy the API key generated (usually starts with `AIzaSy`) and paste it into the Funnelcraft signup screen.

O

OpenRouter

Access multiple open LLM architectures (like Llama-3) with free-tier access keys.

Open OpenRouter ↗
1

Navigate to OpenRouter.ai and create an account or sign in.

2

Click on your account profile avatar on the top-right and select **Keys**, or visit https://openrouter.ai/keys directly.

3

Click the **"Create Key"** button. Name the key (e.g. `Funnelcraft`) and click generate.

4

Copy the secret key immediately (it starts with `sk-or-`). *Note: You will not be able to view this key again!*

GR

Groq Cloud Console

Extreme speed execution of open model layers with standard free usage allowances.

Open Groq Console ↗
1

Sign up or sign in at the Groq Console.

2

Click on the **"API Keys"** item in the left menu, or go directly to https://console.groq.com/keys.

3

Click the **"Create API Key"** button. Label it (e.g. `Funnelcraft`) and submit.

4

Copy the API key generated (it starts with `gsk_`) and save it in Funnelcraft.
